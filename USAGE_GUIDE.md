# 学信档案系统使用说明

## 系统概述

本系统是一个学信档案管理应用，用于查看和管理个人的学籍、学历、学位和考研信息。系统提供了直观的界面和便捷的操作方式，让您可以轻松管理所有教育相关信息。

## 主要功能

### 1. 信息展示

系统首页分为四个主要板块：

- **学籍信息**：显示您的在读学籍记录
- **学历信息**：显示您已获得的学历证书信息
- **学位信息**：显示您已获得的学位证书信息
- **考研信息**：显示您的考研相关信息

每个板块标题后的括号内数字表示该类信息的数量。

### 2. 查看详情

**操作方式**：轻触（单击）任意信息卡片

**功能说明**：
- 进入该条信息的详细页面
- 查看完整的详细信息
- 可以在详细页面中编辑各项信息
- 可以上传相关照片

### 3. 信息卡片操作（长按功能）

**操作方式**：长按（0.5秒）任意信息卡片

**功能说明**：长按后会弹出操作菜单，提供以下三个选项：

#### 3.1 编辑信息
- 修改当前卡片的学校名称、专业、学习形式、学历层次等信息
- 修改后点击保存即可更新信息
- 系统会显示"信息已更新"的提示

#### 3.2 添加新信息
- 在当前类别下快速添加一条新的信息记录
- 新记录会以默认信息创建（学校名称：新学校，专业：新专业）
- 添加成功后，该类别的数量会自动增加
- 系统会显示"已添加新记录"的提示
- 您可以随后编辑新记录以填入真实信息

#### 3.3 删除此条
- 从系统中删除当前选中的信息记录
- 删除后，该类别的数量会自动减少
- 系统会显示"已删除记录"的提示
- **注意**：删除操作不可撤销，请谨慎操作

### 4. 信息绑定

在每个板块的右上角都有"还有XXX没有显示出来？尝试绑定"的链接，点击可以绑定更多的信息记录到系统中。

## 操作技巧

1. **快速查看**：单击卡片可快速查看详细信息
2. **批量管理**：通过长按可以快速添加多条记录
3. **信息整理**：定期删除不需要的记录，保持信息整洁
4. **信息编辑**：既可以在首页长按编辑，也可以进入详情页编辑

## 使用场景示例

### 场景一：添加新的学位信息
1. 长按学位信息板块中的任意一条记录
2. 在弹出菜单中选择"添加新学位信息"
3. 系统自动创建一条新记录
4. 再次长按新记录，选择"编辑信息"
5. 填入真实的学位信息并保存

### 场景二：删除过期的学籍信息
1. 长按要删除的学籍信息卡片
2. 在弹出菜单中选择"删除此条"
3. 确认删除后，该记录从系统中移除

### 场景三：修正信息错误
1. 长按需要修改的信息卡片
2. 在弹出菜单中选择"编辑信息"
3. 修改错误的字段
4. 点击保存完成修正

## 注意事项

- 长按时间需要保持0.5秒以上才能触发操作菜单
- 删除操作不可撤销，请确认后再操作
- 添加的新记录会使用默认值，记得及时编辑为真实信息
- 各类信息的数量会实时更新，确保准确性
