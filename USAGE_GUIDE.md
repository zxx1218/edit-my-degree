# 学信档案系统使用说明

## 系统概述

本系统是一个学信档案管理应用，用于查看和管理个人的学籍、学历、学位和考研信息。系统提供了直观的界面和便捷的操作方式，让您可以轻松管理所有教育相关信息。

## 主要功能

### 1. 信息展示

系统首页分为四个主要板块：

- **学籍信息**：显示您的在读学籍记录
- **学历信息**：显示您已获得的学历证书信息
- **学位信息**：显示您已获得的学位证书信息
- **考研信息**：显示您的考研相关信息

每个板块标题后的括号内数字表示该类信息的数量。

### 2. 查看详情

**操作方式**：轻触（单击）任意信息卡片

**功能说明**：
- 进入该条信息的详细页面
- 查看完整的详细信息
- 在详细页面中直接点击任意字段进行编辑
- 可以上传相关照片（点击照片区域即可上传）
- **重要提示**：在详情页面修改的信息会实时保存，返回主页面后修改仍然保留（刷新页面前修改都会保留）

### 3. 信息卡片操作（长按功能）

**操作方式**：长按（0.5秒）任意信息卡片

**功能说明**：长按后会弹出操作菜单，提供以下三个选项：

#### 3.1 编辑信息
- 修改当前卡片的学校名称、专业、学习形式、学历层次等基本信息
- 修改后点击保存即可更新信息
- 系统会显示"信息已更新"的提示
- 修改会立即保存，返回主页面后仍然保留

#### 3.2 添加新信息
- 在当前类别下快速添加一条新的信息记录
- 新记录会以默认信息创建（学校名称：新学校，专业：新专业）
- 添加成功后，该类别的数量会自动增加
- 系统会显示"已添加新记录"的提示
- 您可以随后点击新记录进入详情页编辑完整信息

#### 3.3 删除此条
- 从系统中删除当前选中的信息记录
- 删除后，该类别的数量会自动减少
- 系统会显示"已删除记录"的提示
- **注意**：删除操作不可撤销，请谨慎操作

### 4. 信息绑定

在学籍、学历、学位信息板块的右上角都有"还有XXX没有显示出来？尝试绑定"的链接，点击可以绑定更多的信息记录到系统中。

## 详细页面编辑说明

### 学籍信息详情页
- **照片上传**：可上传录取照片和学历照片，点击对应照片区域即可选择上传
- **可编辑字段**：姓名、性别、出生日期、学校、专业、学习类型、学历层次、民族、证件号码、学制、学历类别、分院、系所、班级、学号、入学日期、学籍状态、离校日期

### 学历信息详情页
- **照片上传**：可上传个人照片
- **可编辑字段**：姓名、性别、出生日期、学校、专业、学习类型、学历层次、入学日期、毕业日期、学历类别、学制、毕业状态、校长姓名、证书编号

### 学位信息详情页
- **照片上传**：可上传个人照片
- **可编辑字段**：姓名、性别、出生日期、学校、学位类型、学位层次、获学位日期、学科/专业、学位证书编号

### 考研信息详情页
- **照片上传**：可上传个人照片
- **可编辑字段**：包括姓名、学校、年份、考点信息、报名号、报考单位、院系、专业、研究方向、考试方式、专项计划、各科目名称与成绩、总分、录取单位、录取专业、备注等所有显示的信息

**编辑技巧**：
1. 点击任意字段即可进入编辑模式
2. 输入完成后可按Enter键保存，或点击输入框外区域自动保存
3. 按Esc键可取消编辑
4. 每次修改都会实时保存，无需担心数据丢失

## 数据持久化说明

**重要提示**：
- 在详情页面中的所有修改都会实时保存
- 在主页面长按编辑、添加或删除的操作也会立即生效
- **只要不刷新页面，所有修改都会保留**
- 从详情页面返回主页面后，修改的内容仍然存在
- 刷新页面后会恢复到默认数据

## 操作技巧

1. **快速查看**：单击卡片可快速查看详细信息
2. **快速编辑**：进入详情页后直接点击要修改的字段即可编辑
3. **批量管理**：通过长按可以快速添加多条记录
4. **信息整理**：定期删除不需要的记录，保持信息整洁
5. **双重编辑方式**：既可以在首页长按编辑基本信息，也可以进入详情页编辑完整信息

## 使用场景示例

### 场景一：添加新的学位信息
1. 长按学位信息板块中的任意一条记录
2. 在弹出菜单中选择"添加新学位信息"
3. 系统自动创建一条新记录
4. 点击新记录进入详情页
5. 逐个点击字段填入真实的学位信息
6. 返回主页面，新信息已保存

### 场景二：修改详细信息
1. 点击要修改的信息卡片进入详情页
2. 点击需要修改的字段（如学校名称）
3. 输入新的内容
4. 按Enter或点击其他区域保存
5. 返回主页面，修改已生效

### 场景三：删除过期的学籍信息
1. 长按要删除的学籍信息卡片
2. 在弹出菜单中选择"删除此条"
3. 确认删除后，该记录从系统中移除

### 场景四：上传照片
1. 点击信息卡片进入详情页
2. 点击照片区域
3. 选择要上传的照片文件
4. 照片立即显示并保存

## 注意事项

- 长按时间需要保持0.5秒以上才能触发操作菜单
- 删除操作不可撤销，请确认后再操作
- 添加的新记录会使用默认值，记得及时编辑为真实信息
- 各类信息的数量会实时更新，确保准确性
- 所有修改在刷新页面前都会保留
- 详情页面中的所有字段都可以点击编辑
