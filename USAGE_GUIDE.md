# 📚 学信档案系统使用说明

## 📖 系统概述

本系统是一个**学信档案管理应用**，用于查看和管理个人的学籍、学历、学位和考研信息。系统提供了直观的界面和便捷的操作方式，让您可以轻松管理所有教育相关信息。

---

## ✨ 主要功能

### 1️⃣ 信息展示

系统首页分为**四个主要板块**：

| 板块 | 说明 |
|------|------|
| 🎓 **学籍信息** | 显示您的在读学籍记录 |
| 📜 **学历信息** | 显示您已获得的学历证书信息 |
| 🎖️ **学位信息** | 显示您已获得的学位证书信息 |
| 📝 **考研信息** | 显示您的考研相关信息 |

> 💡 **提示**：每个板块标题后的括号内数字表示该类信息的数量。

### 2️⃣ 查看详情

#### 📱 操作方式
**轻触（单击）** 任意信息卡片

<div align="center">
  <img src="/images/usage-tap-gesture.png" alt="单击查看详情" width="300"/>
  <p><em>图2：单击卡片查看详情</em></p>
</div>

#### 🎯 功能说明
- ✅ 进入该条信息的详细页面
- ✅ 查看完整的详细信息
- ✅ 可以在详细页面中编辑各项信息
- ✅ 可以上传相关照片

---

### 3️⃣ 信息卡片操作（长按功能）

#### 📱 操作方式
**长按（0.5秒）** 任意信息卡片

<div align="center">
  <img src="/images/usage-longpress-gesture.png" alt="长按操作示意" width="300"/>
  <p><em>图3：长按卡片触发操作菜单</em></p>
</div>

#### 🎯 功能说明
长按后会弹出操作菜单，提供以下**三个选项**：

<div align="center">
  <img src="/images/usage-longpress-menu.png" alt="长按菜单" width="300"/>
  <p><em>图4：长按后的操作菜单</em></p>
</div>

---

#### 🔧 选项一：编辑信息

<div align="center">
  <img src="/images/usage-edit-dialog.png" alt="编辑对话框" width="300"/>
  <p><em>图5：编辑信息对话框</em></p>
</div>

- ✏️ 修改当前卡片的学校名称、专业、学习形式、学历层次等信息
- 💾 修改后点击保存即可更新信息
- ✅ 系统会显示"信息已更新"的提示

#### ➕ 选项二：添加新信息
- 📝 在当前类别下快速添加一条新的信息记录
- 🆕 新记录会以默认信息创建（学校名称：新学校，专业：新专业）
- 📈 添加成功后，该类别的数量会自动增加
- ✅ 系统会显示"已添加新记录"的提示
- 💡 您可以随后编辑新记录以填入真实信息

#### 🗑️ 选项三：删除此条
- ❌ 从系统中删除当前选中的信息记录
- 📉 删除后，该类别的数量会自动减少
- ✅ 系统会显示"已删除记录"的提示
- ⚠️ **注意**：删除操作不可撤销，请谨慎操作

---

### 4️⃣ 信息绑定

在每个板块的右上角都有 **"还有XXX没有显示出来？尝试绑定"** 的链接，点击可以绑定更多的信息记录到系统中。

## 💡 操作技巧

| 技巧 | 说明 |
|------|------|
| 🔍 **快速查看** | 单击卡片可快速查看详细信息 |

## 📋 使用场景示例

### 🎯 场景一：添加新的学位信息
```
步骤 1️⃣ → 长按学位信息板块中的任意一条记录
步骤 2️⃣ → 在弹出菜单中选择"添加新学位信息"
步骤 3️⃣ → 系统自动创建一条新记录
步骤 4️⃣ → 再次长按新记录，选择"编辑信息"
步骤 5️⃣ → 填入真实的学位信息并保存
```

### 🎯 场景二：删除过期的学籍信息
```
步骤 1️⃣ → 长按要删除的学籍信息卡片
步骤 2️⃣ → 在弹出菜单中选择"删除此条"
步骤 3️⃣ → 确认删除后，该记录从系统中移除
```

### 🎯 场景三：修正信息错误
```
步骤 1️⃣ → 长按需要修改的信息卡片
步骤 2️⃣ → 在弹出菜单中选择"编辑信息"
步骤 3️⃣ → 修改错误的字段
步骤 4️⃣ → 点击保存完成修正
```

---

## ⚠️ 注意事项

> ⏱️ **长按时间**：需要保持 **0.5秒** 以上才能触发操作菜单
> 
> 🚫 **删除警告**：删除操作 **不可撤销**，请确认后再操作
> 
> 📝 **默认值提醒**：添加的新记录会使用默认值，记得及时编辑为真实信息
> 
> 🔄 **实时更新**：各类信息的数量会实时更新，确保准确性
> 
> 🔐 **数据安全**：所有个人信息包括上传的照片都会加密后进行数据持久化，下次登录后依旧存在
